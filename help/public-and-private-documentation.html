# Public & Private documentation

By default your documentation is private. This means that only you logged in users that are in your team can access it. If you want to make your documentation public, you can do so by following the steps below.

# How to make your documentation public

  1. Go to the Apiation dashboard (opens new window)
  2. Select the host you want to make public
  3. Click on three dots on the top right
  4. Click the Make public button

# How to make your documentation private

  1. Go to the Apiation dashboard (opens new window)
  2. Select the host you want to make private
  3. Click on three dots on the top right
  4. Click the Make private button
